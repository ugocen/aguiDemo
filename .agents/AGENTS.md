# Antigravity Workspace Rules

## Collaboration Protocol Adherence
- **ALWAYS** start every task by running `git pull` to ensure you have the latest changes from other agents.
- **ALWAYS** end every task by updating `resources/HANDOFF.md` with your work log. THIS IS A STRICT RULE AND MUST NOT BE SKIPPED.
- **ALWAYS** commit the changes, and run `git push` after updating the work log.
- Multiple agents are working in this workspace, and failure to pull/push will lead to conflict and lost work. Do not skip these steps under any circumstance!
